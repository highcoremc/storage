# Storage
Simple storage component-abstraction for paper/bungee minecraft plugins.

## How to use it?

```xml
<repositories>
  <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
  </repository>
</repositories>
```

```xml
<dependency>
    <groupId>com.github.loperd</groupId>
    <artifactId>storage</artifactId>
    <version>1.4.1</version>
</dependency>
```
